Manual of **AMS** (Assignment Management System)
===

## Functions

1. Signin and signup.
2. Upload and download files.
3. Teacher can manage all students' assignments
4. Student can manage ones' own assignment

## Modify

1. 给上传文件添加撤销的功能
2. 学生成绩统计功能的完善
3. 当前的老师或者学生可以查看所有课程
    * 需要添加"学生-课程"表
    * "课程-课程号"表中添加"教师"字段
    * 当前默认学生可以选择所有课程
4. 没有处理文件名太长或者文件太大的问题
